# FitD Tattered King Harm Tracker

A Foundry VTT module for the Tattered King campaign that adds harm and trauma tracking utilities for the Blades in the Dark system.

## Requirements
- Foundry VTT v14
- Blades in the Dark system v6.1+
- Blades in the Dark Alternate Sheets module

## Installation
1. Copy the `fitd-ttk-harm` folder into your Foundry `Data/modules` directory.
2. Enable the module in the Foundry module configuration screen.
3. Make sure the Blades in the Dark system and the alternate sheets module are installed and enabled.

## Notes
This module provides a starter skeleton for custom harm logic. Add your custom behavior under `scripts/harm.js`.
